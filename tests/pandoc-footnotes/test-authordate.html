Footnote and citation interleaving test

Beginning

This opening paragraph carries my own hand-written aside right away1. It exists to claim the first footnote slot before any citation appears, so we can see whether citeproc inserts generated notes ahead of it or after it.

Middle: a lone citation

Here a Pandoc citation stands by itself with no hand-written marker nearby (Aldgate 2019). Under a notes style this should generate a footnote; under author-date it should stay inline. Nothing of mine competes for this slot.

Same-sentence edge case

This is a claim with my own aside2 and also a citation (Bromfield 2021), placed immediately adjacent so we can see which note number lands first when a hand-written marker and a citation share one sentence.

Reference-before-definition edge case

This sentence references a footnote3 whose definition is written far below, after the citation(Aldgate 2019) that follows it here, to test whether numbering follows reference order or definition order in the source.

End

A final hand-written footnote closes the document4 to confirm the last slot in the merged sequence.
