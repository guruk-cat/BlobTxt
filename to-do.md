# User Config for Merging Headings

## Per-blob config

A blob and the headings therein are shown as a group. Within a group, the following are available:

- Highest level of heading in the blob
- If there are several headings that share the highest level, how many there are
- Option to "demote" the headings by N levels
- If blob doesn't have any headings, option to provide one (heading content and level)

## Merge-wide config

Additional operations are available that affect all the blobs being merged:

- Demote *all* headings by N levels
- Renumber all headings (nested number system; 1. Blah, 1.1. Blah, 1.1.1. Blah, etc.)
- Option to number the H1 heading(s). Off by default.
